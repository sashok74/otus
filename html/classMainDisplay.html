Detailed Description

Конкретная реализация интерфейса IDisplay для отображения примитивов документа.

Этот класс предоставляет реализацию метода show для отображения имен примитивов, содержащихся в объекте Document.

@method void show(Document *doc)

Отображает имена примитивов в данном документе.

Здесь возможно реализовать более сложную логику отображения, например, графический интерфейс. другой вариант - вывод в консоль.

Parameters
docУказатель на объект Document, примитивы которого должны быть отображены. Если документ равен нулю, метод ничего не делает.
