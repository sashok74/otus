Detailed Description

Класс для вывода IP-адресов, представленных в различных форматах.

IpPrinter предоставляет шаблонный метод для форматирования и вывода IP-адресов, представленных в различных типах данных, таких как целочисленные типы, строки, контейнеры (например, std::vector<int>, std::list<short>) и кортежи (std::tuple) с однотипными элементами.

Выводит IP-адрес в формате, соответствующем типу входного значения.

Шаблонный метод поддерживает следующие типы:

  • Целочисленные типы (int, long, uint32_t и т.д.): выводятся в формате IPv4, где каждые 8 бит преобразуются в байт и разделяются точкой.
  • std::string: выводится как строка с пояснением.
  • Контейнеры (std::vector<int> или std::list<short>): значения выводятся через точку.
  • std::tuple с однотипными элементами: значения кортежа выводятся через точку.
